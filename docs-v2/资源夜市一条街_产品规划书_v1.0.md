## 一、品牌定位

**全称：** 资源夜市一条街
**一句话定位：** 让资源找到你，让你找到资源

**品牌主张：**
> 每一条街都有它的故事，资源夜市一条街——汇聚行业人脉、政企资源、金融服务、项目协同的精准对接平台。

**差异化定位：** 资源入口 + 信任建立 + 内容分发 + 合规桥接

## 二、用户画像与入口

### 目标用户

| 人群 | 痛点 | 诉求 |
|------|------|------|
| 中小企业主 | 缺渠道、缺人脉、缺信息 | 找资金、找项目、找合作 |
| 居间人/掮客 | 手握资源但分散、无平台展示 | 标准化、规模化、安全合规 |
| 投资人/机构 | 项目来源分散、质量参差 | 优质标的、风险过滤 |
| 政府/国企人员 | 想对接民企但渠道有限 | 合规、克制、信任 |

### 流量入口（由外到内）

```
短视频/直播引流
       ↓
搜索SEO（百度/微信）
       ↓
[ 首页 ] ← 建立第一信任
       ↓
[ 免费浏览 ] 脱敏摘要
       ↓
[ 注册/认证 ] 成为会员
       ↓
[ 会员升级 ] 解锁完整信息
       ↓
[ 三种触达 ] 会员独享 / 邮箱触达 / 人工桥接
       ↓
[ 项目推进 ] 进入居间协同流程
```

### 四大首页入口卡片（核心模块）

#### 1. 行业群码
- **入口标题：** 行业群码
- **入口副标题：** 低门槛浏览，入群即资源
- **展示内容：** 行业分类标签（制造/建筑/金融/科技等），点击后展示群码
- **双通道机制：** 入群码（用户扫码）+ 群主码（加群主微信，防失效）
- **权益：** 普通会员每日限 2 条完整信息

#### 2. 政企资源
- **入口标题：** 政企资源
- **入口副标题：** 汇聚机关人脉与企业人脉
- **展示内容：**
  - 机关人脉：各部委/地方政府人脉资源，按地域+领域分类
  - 企业人脉：企业高管资源，按行业+地区分类
- **触达方式：** 会员独享 → 邮箱触达 → 人工桥接（渐进式）

#### 3. 最新求助
- **入口标题：** 最新求助
- **入口副标题：** 真实需求，即时响应
- **展示内容：** 用户/企业主动发布的资源需求请求
- **权益：** 普通会员每日限 2 条，高级/尊享会员不限

#### 4. 居间服务（核心差异化）
- **入口标题：** 居间协同
- **入口副标题：** 护航每一次资源对接
- **展示内容：** U型链路动画演示（甲乙双方 + 居间护航）
- **引荐机制：** 层层接力，公开透明
- **权益：** 尊享会员专享

## 四、会员体系

### 会员等级

| 等级 | 价格 | 权益 |
|------|------|------|
| 普通会员 | 99元/年 | 每日限2条完整信息 |
| 高级会员 | 199元/年 | 不限次数浏览 |
| 尊享会员 | 299元/年 | 不限+居间协同+更多桥接机会 |

### 触达机制（三档）

| 触达方式 | 说明 |
|----------|------|
| 方式一 | 会员独享：直接解锁查看完整联系信息 |
| 方式二 | 邮箱触达：平台代发匿名邮件，双向保密 |
| 方式三 | 人工桥接：平台客服介入，撮合碰面 |

## 五、产品路线图

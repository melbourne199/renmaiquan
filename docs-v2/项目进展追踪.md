## 重要教训记录

> 来自实践的总结，避免重蹈覆辙

1. **分支末尾忘记 return 导致 fall-through** — 多人协作时代码合并需特别注意
2. **球面扇形展开公式：** `tangentX = Y轴.cross(normal).normalize()` — 3D渲染相关
3. **岛礁数据统一用 controlledBy 字段**，不用 isChinaIsland
4. **glow 大小用固定值**（城市0.014，港澳台0.018），不随zoom变化

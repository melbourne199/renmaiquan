## 一、居间协同原理

### 什么是居间协同

资源夜市一条街的居间协同机制，是平台的差异化核心能力。

**核心价值：** 居间方作为甲乙双方之间的"护航者"，提供流程保障、信息透明、争议协调，让每一次资源对接都有据可依、有序推进。

### U型链路动画

```
  甲方（资源提供方）
       ↓
  【一级引荐人】第一时间接住资源
       ↓
  【二级引荐人】继续桥接传递
       ↓
       …
       ↓
  乙方（资源需求方）
```

**视觉设计：** U型链路动画——左侧"岸"=甲方（资源来向），右侧"岸"=乙方（资源去向），底部=居间方（协同护航）

**动画要点：**
- 两束光从左右两岸向中心汇聚
- 中心弹出居间方 Logo
- 配合粒子拖尾效果
- 居间方居中，视觉权重最高

## 二、引荐人机制

### 角色定义

| 角色 | 定义 |
|------|------|
| 一级引荐人 | 最先接住资源/需求，并录入系统的人 |
| 二级引荐人 | 接过资源继续桥接的接力者 |
| N级引荐人 | 链式传递中的第N位桥接者 |

### 分享与录入规则

1. 用户在平台发布资源或需求
2. 系统生成**专属引荐链接**（含用户ID标识）
3. 链接通过社交渠道传播
4. 每位新进入者自动绑定其上一级引荐人
5. 居间方全程可见链路，参与协同

### 激励机制

| 引荐层级 | 说明 |
|----------|------|
| 直接引荐（一级） | 引荐人获得对应项目优先知情权 |
| 间接引荐（二级+） | 引荐人获得协同积分/优先对接机会 |

### 引荐人的权利与义务

**权利：**
- 获取所引荐项目的进展通知
- 在居间协议中作为证人/协调人
- 获得平台服务费分成（如适用）

**义务：**
- 对甲乙双方信息保密
- 不得恶意撮合损害任一方利益
- 及时更新项目进展

## 三、引荐人附加协议（法律文本）

> 以下为平台与引荐人之间的正式协议文本，发布前须经法律顾问审核。

---

**《资源夜市一条街引荐人服务协议》**

**甲方（居间平台）：** 资源夜市一条街
**乙方（引荐人）：** [姓名/企业名称]

**第一条 服务内容**

乙方作为甲方的引荐人，在乙方引荐的项目中，为甲乙双方提供信息传递、材料协调、进度跟进等居间服务。

**第二条 保密义务**

2.1 乙方对在服务过程中获知的甲方及项目相关信息负有严格保密义务，不得向任何第三方泄露。

2.2 保密期限：自协议签署之日起至项目终止后两年（2年）。

**第三条 禁止行为**

乙方承诺不从事以下行为：
- 私自截留甲乙双方往来款项
- 伪造、篡改项目材料
- 以甲方名义对外作出任何承诺或签署文件
- 恶意撮合交易，损害任一方合法权益

**第四条 违规责任与追诉**

4.1 如乙方违反本协议，甲方有权：
- 立即终止乙方的引荐人资格
- 冻结乙方在平台的所有收益
- 将乙方列入平台黑名单，终身禁入

4.2 如因乙方违规行为给甲方或第三方造成损失，甲方保留追究乙方法律责任的权利，包括但不限于：
- 民事赔偿
- 向行业协会或司法机关通报

4.3 违规情节严重者，甲方将协助受损方依法追究乙方的刑事责任。

**第五条 收益分配**

引荐服务费按项目类型在专项协议中约定，甲方有权收取平台服务费，剩余部分按引荐链条分配。

**第六条 争议解决**

因本协议引起的任何争议，双方应首先协商解决；协商不成的，提交[甲方所在地]人民法院管辖。

## 五、操作流程

### 资源发布流程

```
1. 用户注册 → 2. 选择类型（群码/政企/求助/居间）
   → 3. 填写信息（标题/内容/分类/地域/联系方式）
   → 4. AI辅助优化 → 5. 审核发布
   → 6. 自动生成引荐链接
```

### 居间协同流程

```
1. 甲乙双方达成初步合作意向
   → 2. 任一方发起居间申请
   → 3. 平台指派引荐人/居间方
   → 4. 居间方录入U型链路
   → 5. 各方签署电子协议
   → 6. 居间方跟进进度，定期更新
   → 7. 项目完成，居间结算
```

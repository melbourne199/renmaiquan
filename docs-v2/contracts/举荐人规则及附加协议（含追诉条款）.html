📢 重要法律声明：关于追诉对象的特别约定

1. 甲方/乙方支付责任限制：甲方/乙方的支付义务仅限于按照本协议第三条第1款将居间服务费总额支付至指定监管账户。一旦完成此支付，甲方/乙方对款项的后续分配不承担任何责任。

2. 禁止不当追诉：任何举荐人不得以任何理由将甲方或乙方作为分配纠纷的共同被告或第三人，除非能证明甲方/乙方故意不支付或欺诈性转移居间服务费总额。

3. 错误追诉赔偿：若任何举荐人违反本约定，在甲方/乙方已履行总额支付义务的情况下仍将其列为被告或第三人，该举荐人需赔偿甲方/乙方因此产生的全部损失（含律师费、诉讼费、误工损失等），并承担不低于索赔金额30%的违约金。

4. 法院移送与费用承担：如发生此类错误诉讼，甲方/乙方有权申请法院将案件移送至本协议约定的正确管辖法院，且由此产生的全部费用由错误起诉方承担。

附件一：举荐团队双级分配规则

本项目中，凡参与项目机会传递、资源引荐、信息沟通及辅助推进的人员，统一称为“举荐人”，不再使用“中间人”称谓。
本规则以角色清晰、分配透明、事前认同为原则，不涉及身份高低，仅基于实际所处位置与核心价值进行区分。

一、角色分级

一级举荐人：指与甲方或乙方拍板决策人存在直接私人关系，并能够将项目机会直接触达决策层的关键人员。
甲方侧：一级资源人   |   乙方侧：一级居间人

二级举荐人：除一级举荐人之外，所有通过举荐链条参与项目的其他人员，负责信息传递、关系扩散及落地支持等工作。

二、居间费分配规则

设本项目居间服务费总额为T（T为主合同约定的居间服务费总额）。
一级分配：占总额60%，由两名一级举荐人共享，原则上各占30%，如另有书面约定则按约定执行。
二级分配：占总额40%，由全部二级举荐人平均分配，不因人数变化而调整比例。
若无法认可本规则，可在项目开始前自行选择退出，不影响规则整体执行。

附件二：关于举荐人链条确认及支付责任的附加协议

第一条 协议目的

为保障所有实际参与并产生贡献的举荐人合法权益，明确各方责任，确保居间服务费及时、准确分配，特制定本附加协议。本协议作为主合同的附件，与主合同具有同等法律效力。

第二条 举荐人身份确认

1. 本项目所有举荐人身份需在项目启动前由一级举荐人共同确认并书面记录。
2. 二级举荐人名单应在第一笔居间服务费支付前最终确定，后续不再增减。
3. 每个举荐人需提供真实有效的身份信息及收款账户，因信息错误导致的支付问题由本人承担。
4. 举荐人链条清单（见附件三）为本协议组成部分，与协议正文具有同等法律效力。

第三条 支付责任与流程

1. 甲方应按照主合同约定的居间服务费支付条款，在收到乙方支付款项后3个工作日内，将居间服务费总额T支付至一级举荐人指定的共同监管账户，并向所有举荐人发送付款凭证复印件。
2. 甲方完成上述支付后，其在本协议项下的支付义务即告完全履行完毕，不对后续分配承担任何责任。
3. 一级举荐人作为分配执行人，应在收到全款后5个工作日内完成分配，并向所有举荐人发送分配明细表。
4. 分配流程：优先支付二级举荐人分配总额（40% T）平均分配给所有二级举荐人，剩余部分（60% T）由两名一级举荐人按约定比例分配。
5. 支付凭证的确认：甲方完成付款后3个工作日内，应通过电子邮件或微信等可留存记录的方式，向所有举荐人提供银行转账凭证截图，作为支付完成的证明。

第四条 错误追诉的防范与责任（核心保护条款）

4.1 责任隔离机制：各方明确确认，甲方/乙方的责任仅限于将居间服务费总额支付至指定账户。款项进入监管账户后，即视为甲方/乙方完全履行了支付义务。

4.2 禁止滥用诉权：任何举荐人不得以以下理由将甲方/乙方列为追诉对象：一级举荐人未及时分配款项；一级举荐人分配不公；一级举荐人挪用、侵占款项；其他与甲方/乙方总额支付义务无关的事由。

4.3 错误追诉的严重后果：若任何举荐人违反上述约定，在甲方/乙方已提供付款凭证的情况下仍将其列为被告，则：
- 该举荐人需在起诉后15日内向甲方/乙方支付维权保证金人民币50,000元；
- 如甲方/乙方胜诉或案件被驳回，该保证金不予退还，作为甲方/乙方的部分补偿；
- 该举荐人需承担甲方/乙方因此产生的全部律师费、诉讼费、保全费、差旅费及误工损失；
- 该举荐人需向甲方/乙方支付不低于其索赔金额30%的违约金。

4.4 反诉与追偿权：如发生错误追诉，甲方/乙方有权立即提起反诉，要求赔偿全部损失；向行业组织、合作伙伴通报该举荐人的不当诉讼行为；在未来所有项目中拒绝与该举荐人合作。

4.5 恶意诉讼的刑事风险提示：若举荐人明知甲方/乙方无责任仍提起诉讼，企图通过诉讼施压获取不当利益，可能构成虚假诉讼或敲诈勒索，甲方/乙方保留向公安机关报案的权利。

第五条 款项未收到时的正确追诉程序

1. 若任何举荐人未收到款项，应首先核对甲方是否已提供付款凭证。如甲方已提供凭证，则追诉对象仅限一级举荐人。
2. 如甲方未提供付款凭证或凭证显示未付款，举荐人可先书面催告甲方履行总额支付义务。
3. 一级举荐人承担连带支付责任，未收到款项的举荐人有权向任意一名或两名一级举荐人追索。
4. 具体追诉流程：书面催告（7日）→向一级举荐人提起诉讼→要求支付欠款及违约金（每日千分之一）。
5. 在任何情况下，不得以“施压”、“加快推进”等为由将甲方/乙方列为共同被告，否则承担本协议第四条规定的全部违约责任。

第六条 链条清单的法律效力

1. 附件三《举荐人链条清单》为本协议核心附件，清单中列明的举荐人为有权参与分配的唯一合法主体。
2. 链条清单的确认需经各方签字或盖章，任何口头承诺或未经确认的加入均无效。
3. 链条清单确认后，任何修改需经全体一级举荐人及受影响方书面同意。
4. 如发生争议，链条清单作为确定各方权利义务的首要依据。

第七条 保密义务

1. 所有举荐人应对本协议内容、分配比例、个人所得等敏感信息严格保密。
2. 不得向任何与本项目无关的第三方透露举荐人链条结构及具体分配方案。
3. 保密义务持续至本项目完全结束且所有款项分配完成后两年。

第八条 争议解决

1. 因本协议引起的或与本协议有关的任何争议，各方应友好协商解决。
2. 协商不成的，任何一方均有权向主合同约定的管辖法院提起诉讼。
3. 管辖权异议与移送：如任何举荐人违反本协议第四条规定，在错误管辖法院起诉，其他方有权提出管辖权异议，要求移送至本协议约定的法院，且异议方不因此丧失其他实体权利。
4. 律师费转付：败诉方需承担胜诉方全部合理的律师费、诉讼费、保全费、鉴定费等维权成本。

第九条 合同份数与电子档

1. 本合同一式____份，其中甲方、乙方各执一份，一级举荐人（2人）各执一份，二级举荐人代表执一份，见证人留存一份，具有同等法律效力。
2. 本合同签署后七个工作日内，甲方（或一级举荐人指定人员）应向所有举荐人提供一份与本协议纸质版内容完全一致的电子档（PDF格式），发送至各举荐人指定邮箱，以供存档和备查。
3. 电子档与纸质原件具有同等法律效力。
4. 所有举荐人均有义务妥善保管本合同电子档及纸质原件，不得遗失、泄露或用于本协议约定之外的其他用途。

第十条 其他约定

1. 本协议作为主合同的附件，与主合同具有同等法律效力。
2. 独立责任条款：本协议第四条的“错误追诉责任”为独立条款，无论本协议其他条款是否有效，该条款始终有效。
3. 司法认知条款：各方同意，在可能发生的诉讼中，可将本协议第四条内容作为证据提交，以证明甲方/乙方的责任范围。
4. 本协议自所有举荐人签字之日起生效，有效期与主合同一致。
